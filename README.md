# Pizza-Sales-Analysis-Dashboard
(Interactive Dashboard creation using MS Excel)

## Project Objective 

The objective of this project is to analyze pizza sales data to gain insights into sales patterns, trends, and customer preferences. Through various visualizations like charts and graphs on the "Pizza Sales Analysis Dashboard", the project aims to provide an understanding of top-selling pizzas, order trends over months, popular pizza sizes and categories, and other key sales metrics. This analysis could help in making informed business decisions related to pizza sales and operations.

## Dataset used
- <a href="https://github.com/vaibhavideshmukh2711/Pizza-Sales-Analysis-dashboard/commit/179122839b7475291bc4b2f1a4b297d455834836">Dataset</a>

## Questions (KPIs)

-  Which pizza type is the top seller according to the line graph?
-  In which month were the number of orders the highest?
-  Which pizza size has the highest order quantity?
-  Which pizza category is the most popular based on the bar chart?
-  Top 5 Pizza Type With Highest Avg price
-  In which month was the count of orders the lowest?
-  Which of the top 10 pizza types has the highest average quantity ordered?
-  Pizza Sales by ID and Size



- Dashboard Interaction <a href="https://github.com/vaibhavideshmukh2711/Pizza-Sales-Analysis-dashboard/blob/main/Dashboardimg.png">View Dashboard</a>

## Process
- Verify data for any missing values and anomalies, and sort out the same.
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created pivot tables according to the questions asked.
- Merge all pivot tables into one dashboard and apply slicer to make dynamic.

## Project Insight

- Top 5 Selling Pizzas: The graph shows the top-selling pizzas,  "The Barbecue Meat Lover" is one of them.
- Orders vs Quantity: Orders and quantity sold vary by month. July has a peak in both orders and quantity sold.
- Order Quantity vs Pizza Size: The pie chart shows distribution of orders by pizza size. One size seems to be the largest portion (likely the most popular size).
- Most Popular Pizza Category: Classic pizzas are the most popular category.
- Top 5 Pizza Types with Highest Avg Price: Shows the top pizzas by average price. "The Greek" seems to be one of them.
- Per Month Count of Order ID: Orders fluctuate monthly, with July having a notable peak.
- Average QTY of Top 10 Pizza Types: Shows average quantity sold for top pizza types.


## Final Conclusion:

The Pizza Sales Analysis Dashboard provides key insights into sales performance and trends. It highlights the top 5 selling pizzas, shows orders and quantity trends by month, and displays the distribution of order quantity across different pizza sizes. Classic pizzas are the most popular category. Additionally, it showcases the top pizzas by average price. Overall, the dashboard helps in understanding sales patterns for better business decisions.
